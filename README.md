# regex-validation

- [main](https://github.com/RafaelHades/regex-validation.git)

این پروژه شامل سه برنچ جداگانه برای اعتبارسنجی با Regex است:

- [phone-regex](https://github.com/RafaelHades/regex-validation/tree/phone-regex) : اعتبارسنجی شماره تلفن همراه با فرمت +98912xxxxxxx
- [national-code-regex](https://github.com/RafaelHades/regex-validation/tree/national-code-regex) : اعتبارسنجی کد ملی ۱۰ رقمی با رقم اول غیرصفر
- [name-regex](https://github.com/RafaelHades/regex-validation/tree/name-regex) : اعتبارسنجی نام (فقط حروف فارسی و انگلیسی و فاصله)

برای مشاهده کد هر بخش، لطفاً به برنچ مربوطه بروید.
با تشکر، حمیدرضا فاضلی
